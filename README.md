# mongo-aggregate-example
How to use aggregation in MongoDB


## SQL to Aggregation Mapping

- `SELECT` = `$project`
- `WHERE` = `$match`
- `ORDER BY` = `$sort`
- `LIMIT` = `$limit`

Reference : [SQL to Aggretation Mapping Chart](https://docs.mongodb.com/manual/reference/sql-aggregation-comparison/)
